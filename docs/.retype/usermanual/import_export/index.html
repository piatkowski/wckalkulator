# Import/Export fieldsets

WC Kalkulator is built on the new Post Type, which is full compatible with core import/export functionality in Wordpress. Navigate to Tools > Export to export Fieldset post type to XML file. Use Tools > Import to import that file.

You can also use any importer/exporter plugin in Wordpress.
